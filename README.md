# Fresh Bites 🍔

Fresh Bites is a full-stack food ordering web application built with React, Node.js, Express, and MySQL.

## Features

- Customer registration and login
- JWT authentication
- Food menu and cart system
- Checkout and order placement
- Wallet balance system
- UPI/Razorpay payment support
- Live order tracking
- Rider auto-assignment
- Admin dashboard
- Admin users, orders, foods, wallet, riders, and analytics management
- MySQL permanent data storage

## Tech Stack

**Frontend:** React, Redux Toolkit, React Router  
**Backend:** Node.js, Express.js  
**Database:** MySQL  
**Authentication:** JWT  
**Payments:** UPI / Razorpay  
**Email:** Nodemailer  

## Project Purpose

This project is developed as a complete online food ordering system where customers can order food, track delivery, and manage wallet balance, while admins can manage users, orders, riders, foods, payments, and analytics.
